# Mobile y Performance - Landing y Login

## 1. Mobile landing
- Cards adaptables sin scroll horizontal forzado.
- Tipografía legible en pantallas chicas.
- CTAs grandes y separados.
- Carrusel táctil-friendly.

## 2. Mobile login
- Inputs con altura mínima 44px.
- Botones full width útiles.
- Fuente de input >= 16px para evitar zoom iOS.
- Espaciado generoso entre campos.

## 3. Performance
- CSS crítico inline.
- Imágenes con lazy load.
- Scripts diferidos en landing.
- Preconnect a API en login.

## 4. Próximos pasos
- Medir FCP y LCP en landing.
- Optimizar tamaño JS bundle.
- Evaluar PWA para app.
